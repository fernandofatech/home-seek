# Usage

To install the app just run:

```
npm i -g home-seek
```

To use this app simply go to Daft, perform your search. Then copy the URL and Run:

```
home-seek -u '<DAFT_SEARCH_URL>'

or if you cloned this repo:

node ./bin/run -u '<DAFT_SEARCH_URL>'
```
